# BMS Weekly Report Skill

BMS 算法追踪周报生成技能 - 自动生成BMS/电池技术周报并推送到微信公众号草稿箱

## 功能

- 自动收集BMS/电池领域的学术论文、厂商动态、开源项目、专利技术、行业标准
- 按信源等级 (T1-T4) 分级标注
- 生成符合微信公众号格式的HTML内容（全内联样式）
- 自动推送到微信公众号草稿箱
- 推送后自动验证（标题、H2数量、URL数量、中文字数、期刊标签数）

## 报告结构

1. **本周趋势展望** (3-5 items)
2. **一、学术论文进展** (10 items)
3. **二、厂商动态** (10 items)
4. **三、开源项目与数据集** (10 items)
5. **四、专利技术** (10 items)
6. **五、行业标准** (10 items)

## 使用方法

将 `bms-weekly-report` 目录放入 TRAE 的 `.trae/skills/` 目录下，然后在 TRAE 中说：

> 用这个技能跑本周的BMS算法周报，推送到我的微信公众号草稿箱

## 信源分级

| Tier | 颜色 | 范围 |
|------|------|------|
| T1 | 绿色 #6abf69 | 顶刊/政府数据/院士实验室 |
| T2 | 蓝色 #4a90d9 | SCI期刊/IEEE/arXiv/权威媒体 |
| T3 | 橙色 #e67e22 | 企业官方/国内核心期刊/行业新闻 |
| T4 | 紫色 #9b59b6 | 行业博客/自媒体 (尽量避免) |

## 注意事项

- 所有日期必须在发布日前7天内
- 禁止使用 toutiao.com 链接
- DOI需通过Crossref验证
- GitHub仓库需确认存在
- 专利号需通过CNIPA验证
- Unicode转义序列需检查是否有hex digit吞噬问题
- 推送编码必须使用 `ensure_ascii=False` + `charset=utf-8`
